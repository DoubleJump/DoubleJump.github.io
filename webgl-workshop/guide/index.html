Practical 3D for Websites

During this workshop we'll be building a 3D product turntable with the aim of covering the basics of rendering and interaction with THREEjs. All the code for this workshop can be found here.

File Structure

A premade folder structure and starter files can be downloaded here. Feel free to rearrange things how you see fit.

Software

To get things started we'll need to make sure we have some software installed. These are just suggestions. If you use alternatives that you are more comfortable with then you should use those.

  • Sublime Text
  • Blender3D
  • Krita
  • Python 2.7

Local Web Server

THREEjs' asset loading functions require a local web server. Set one running by going to the command line, navigate to the workshop folder and type python -m SimpleHTTPServer 8000 Open your browser and enter http://localhost:8000 into the address bar.
